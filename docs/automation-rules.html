Key Formulas

ACOS
adSpend / adRevenue
Target: ≤ 30%  |  Break-even: 45%
TACOS
adSpend / totalRevenue
totalRevenue = adRevenue + organicRevenue
Break-Even ACOS
(price - COGS - FBA - storage) / price
ACOS below this = you make money
CVR (Conversion Rate)
orders / clicks
Healthy: 10-20%+
CTR (Click-Through Rate)
clicks / impressions
Healthy: 0.3-0.5%+
ROAS
adRevenue / adSpend
Inverse of ACOS. Higher = better

Pre-Launch Readiness

Gates that must pass before any PPC spend begins.

R01
Minimum Reviews Gate
CriticalManual
reviews < 15
✗ BLOCK scaling. Enroll in Amazon Vine until reviews ≥ 15.
R02
Target ACOS Calculation
CriticalManual
targetAcos = (price - COGS - FBA_fees - storage) / price
✎ Compute before launching any campaign. Set as TARGET_ACOS in .env.
R03
Keyword List Minimum
HighManual
keywordCount < 30
⚠ WARN: Need 30-50 target keywords before launch. Use Helium10 or DataForSEO.

Campaign Structure

Structural invariants that must always hold. Violations break optimization.

R37
One Match Type Per Ad Group
CriticalAuto
countDistinct(adGroup.matchTypes) > 1
✗ VIOLATION: Split into separate ad groups. Cannot bid-optimize per match type if mixed.
R38
One Product Per Ad Group
CriticalAuto
countDistinct(adGroup.products) > 1
✗ VIOLATION: Split into separate ad groups for product-level ACOS analysis.
R39
Never Move Winning Keywords
HighManual
keyword.isWinner AND keyword.isBeingMoved
✗ BLOCK: Winners lose history in new campaigns. Move the losers out instead.
R18
Cross-Campaign Negative Isolation
HighAuto
keyword.matchType == EXACT
↦ Add as NEGATIVE EXACT in all Auto, Phrase, and Broad campaigns to prevent cannibalization.

Launch Phase (Weeks 1-4)

First 4 weeks after campaign creation. Priority: velocity over profitability.

R04
7-Day No-Touch Window
CriticalAuto
daysSinceLaunch < 7
🔒 BLOCK all bid and budget changes. Algorithm needs 7 days of data.
R05
Auto Campaign Initial CPC
MediumManual
campaign.type == AUTO AND phase == LAUNCH
$ Set CPC = $0.50–$0.75. Daily budget = $20–$50.
R06
TOS Modifier on Ranking Campaigns
HighAuto
campaign.type == RANKING AND phase == LAUNCH
↑ Set Top of Search placement modifier = 50–100%.
R07
High ACOS is Normal During Launch
MediumAuto
phase == LAUNCH AND acos > 45%
✓ NO ACTION — this is a rank investment. Do NOT reduce bids during launch.
5.1

Bid Optimization Decision Tree

Weekly keyword-level decisions. Evaluate top-to-bottom; first match wins.

Decision Tree

Keyword Data
  ├─ impressions == 0?
  │  └─ YES → R08: Raise bid 25%
  ├─ clicks == 0?
  │  ├─ impressions ≥ 100 → R09: Review image & title
  │  └─ impressions < 100 → Needs more data
  ├─ orders == 0?
  │  ├─ clicks ≥ 10 → R10: NEGATE
  │  └─ clicks < 10 → R11: Wait for more data
  └─ Has sales → evaluate ACOS
     ├─ acos > 45% → R12: Lower bid 15%
     ├─ 30% < acos ≤ 45% → R13: Lower bid 7.5%
     ├─ acos ≤ 30% & orders ≥ 3 → R14: Isolate to SKC
     └─ acos ≤ 30% & orders < 3 → R15: Raise bid 20%
R08
No Impressions — Raise Bid
HighAuto
impressions == 0
↑ Raise bid 25%.   newBid = currentBid * 1.25
R09
Impressions, No Clicks — Creative Issue
HighManual
impressions ≥ 100 AND clicks == 0
📷 Do NOT adjust bid. Review main image and title relevance. This is a creative problem.
R10
Clicks, No Sales — Negate Bleeder
CriticalAuto
clicks ≥ 10 AND orders == 0
✗ Add as NEGATIVE EXACT. Pause keyword. Wasted spend.
R11
Clicks, No Sales — Needs Data
LowAuto
clicks > 0 AND clicks < 10 AND orders == 0
⌛ WAIT — need 10 clicks minimum before negating. Monitor next week.
R12
ACOS Above Break-Even — Lower Bid
CriticalAuto
acos > 45%
↓ Lower bid 15%.   newBid = currentBid * 0.85
R13
ACOS Between Target and Break-Even — Gentle Reduction
HighAuto
30% < acos ≤ 45%
↓ Lower bid 7.5% (gentle).   newBid = currentBid * 0.925
R14
Low ACOS, Strong Performer — Isolate to SKC
HighManual
acos ≤ 30% AND orders ≥ 3
★ Isolate into Single Keyword Campaign (SKC). Increase budget. Do NOT remove from original — move losers instead.
R15
Low ACOS, Few Orders — Scale Up
MediumAuto
acos ≤ 30% AND orders < 3
↑ Raise bid 20%.   newBid = currentBid * 1.20
5.2

Negative Keywords

Weekly harvest from Search Term Report. Negate waste immediately.

R16
High Clicks, Zero Sales
CriticalAuto
clicks ≥ 10 AND orders == 0
✗ Add as NEGATIVE EXACT in the source campaign.
R17
ACOS Above Break-Even
CriticalAuto
searchTerm.acos > 45%
✗ Add search term as NEGATIVE EXACT — losing money on every sale from this term.
5.3

Placement Optimization

Top of Search vs. Rest of Search vs. Product Pages — evaluate separately.

R19
TOS Outperforming
MediumManual
tos.acos < rest.acos AND tos.conversions > 0
↑ Increase TOS placement modifier (up to 100%).
R20
Product Pages Converting — Don't Pause
HighManual
productPages.orders > 0 AND overall.acos > targetAcos
⚠ Do NOT pause. Keyword looks bad overall but product pages are converting. Adjust placement modifiers, not base bid.

MediumAuto

🌙 Reduce bids during 12am–6am to prevent overnight waste on low-conversion traffic.

Weekly Maintenance

Every Monday morning. Guard rails to prevent over-optimization.

R35
Budget Cap Warning
HighAuto
campaignSpend ≥ dailyBudget * 0.90
⚠ WARN: Hitting budget cap. Missing afternoon/evening peak hours. Increase budget or reallocate.
R36
Minimum Days Between Bid Changes
HighAuto
daysSinceLastBidChange < 7
🔒 BLOCK bid change. Algorithm needs 7+ days of data per change. Frequent changes disrupt learning.

CriticalManual

LowAuto
tacos[w] < tacos[w-1]
✓ Stay in current phase. Strategy is working — do not change approach.

Funnel Diagnostic

When sales stall, find which stage of the funnel is broken.

Impression
R30: bid / keyword
Click
R31: image / title
Add to Cart
R32: listing quality
Purchase
R32: price / reviews
Organic Rank
R33-34: budget / traffic
R30
No Impressions
HighManual
impressions == 0
🔍 FIX: Bid too low or keyword irrelevant. Raise bid 20-30% or swap keyword.
R31
Impressions, No Clicks
HighManual
impressions > 0 AND clicks == 0
📷 FIX: Bad main image or title. CTR problem — improve creative.
R32
Clicks, No Sales
HighManual
clicks > 0 AND orders == 0
🛒 FIX: Listing quality — check images, bullets, price, reviews, A+ content.
R33
Good ACOS, No Organic Lift
MediumManual
acos ≤ 30% AND organicRank NOT improving
🚀 FIX: Not enough velocity. Increase budget. Add external traffic (TikTok, Instagram, Facebook).
R34
Good PPC, Organic Declining
HighManual
ppc.sales > 0 AND organicRank declining
⚠ FIX: Competitor outranking. Increase TOS modifier. Drive external traffic.
