Monday Maintenance Checklist

Every Monday morning. Complete in order. Takes ~30 minutes.

1

Download Reports

Pull Search Term Report (STR), Keyword Performance Report, and Placement Report from Amazon Ads Console.

2

Harvest Negatives

Classify STR. Any term with 10+ clicks and 0 orders → add as Negative Exact. Any term losing money (ACOS > break-even) → negate.

3

Bid Adjustments

Run the decision tree on each keyword. Raise winners. Lower bleeders. Negate lost causes. Isolate proven performers into SKC.

4

Review Placements

Check Top-of-Search vs Product Pages performance. Raise TOS modifier if ACOS ≤ target. Lower if above break-even.

5

Check TACOS

Calculate this week's TACOS. Compare to last week. Is organic growing? Are you ready to phase-shift?

6

Document & Plan

Log changes made. Note keywords promoted to SKC. Flag anything unusual for investigation.
